Project Overview

This project analyzes Swiggy food delivery sales data to understand revenue trends, top-performing cities, and customer food preferences.

Key Analysis

- Revenue by state
- Top 5 cities by sales
- Veg vs Non-Veg revenue contribution
- Daily sales trends
- Quarterly performance summary

Key Insights

- Bengaluru generated the highest sales revenue among all cities.
- Vegetarian dishes contributed more than 60% of total revenue.
- Order volume increased significantly during the second quarter.
- Certain states show stronger food delivery demand compared to others.
